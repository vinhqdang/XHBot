\section{Methodology}

The fundamental objective of the XHBot framework is to formulate social bot detection not merely as standard node classification, but as an adversarial representation learning problem over heterophilic graphs. In this section, we rigorously define the problem space and present the mathematical formulation of our four operational modules.

\subsection{Problem Formulation}
Let the social interaction network be denoted as a heterogeneous graph $\mathcal{G} = (\mathcal{V}, \mathcal{E}, \mathcal{R}, X)$, where $\mathcal{V}$ is the set of $N$ nodes (users), $\mathcal{E}$ is the set of directed edges, and $\mathcal{R}$ is the set of edge relation types (e.g., \texttt{follows}, \texttt{retweets}). Each node $v \in \mathcal{V}$ is associated with a $d$-dimensional semantic feature vector $\mathbf{x}_v \in \mathbb{R}^d$ derived from profile metadata and textual embeddings. The adjacency matrix under relation $r$ is denoted as $A^{(r)} \in \{0, 1\}^{N \times N}$. 

The task is to learn a mapping function $f: (\mathcal{G}, X) \rightarrow \mathcal{Y}$, where $\mathcal{Y} = \{0, 1\}$ represents benign humans and malicious bots, respectively, while maximizing robustness against adversarial relation camouflage and producing interpretable subgraph rationales.

\subsection{Spectral-Guided Topology Refinement (SGTR)}
Camouflaged bots intentionally form cross-class edges to dilute their anomalous signatures. From a spectral graph theory perspective, these cross-class connections manifest as high-frequency noise on the graph Laplacian $L = D - A$, causing over-smoothing during the low-pass filtering inherent to standard GCNs. 

To counteract this, the SGTR module mathematically isolates and penalizes high-frequency (heterophilic) edges prior to message passing. We define the pairwise spectral energy $E^{(r)}(u,v)$ for an edge $e_{u,v} \in \mathcal{E}^{(r)}$ as:
\begin{equation}
    E^{(r)}(u,v) = \sigma \left( \mathbf{W}_{spec} [\mathbf{x}_u \parallel \mathbf{x}_v \parallel (\mathbf{x}_u - \mathbf{x}_v)^2] + \mathbf{b}_{spec} \right)
\end{equation}
where $\parallel$ denotes concatenation, $\sigma$ is the Sigmoid activation function, and $\mathbf{W}_{spec}$ is a learnable projection matrix. A high energy value indicates a severe semantic mismatch typical of bot-to-human camouflage. We apply a soft-pruning threshold $\tau$ to construct a refined adjacency matrix $\tilde{A}$:
\begin{equation}
    \tilde{A}^{(r)}_{u,v} = A^{(r)}_{u,v} \cdot \left( 1 - \mathbb{I}(E^{(r)}(u,v) > \tau) \cdot E^{(r)}(u,v) \right)
\end{equation}
This operation explicitly down-weights adversarial camouflage edges, preserving the low-frequency topological integrity required for accurate homophilic community detection.

\subsection{Tri-Channel Heterophily-Aware Aggregation (THCA)}
Because bots operate in both homophilic clusters (botnets) and heterophilic distributions (camouflage), a single aggregation function is provably insufficient. We introduce THCA, which partitions neighborhood message passing into three mathematically distinct channels to capture orthogonal structural signals.

\textbf{1. Homophilic Channel ($\mathbf{h}^{H}_v$):} Captures cohesive community structures via normalized Laplacian smoothing over the refined graph $\tilde{A}$:
\begin{equation}
    \mathbf{h}^{H}_v = \text{ReLU} \left( \sum_{r \in \mathcal{R}} \sum_{u \in \tilde{\mathcal{N}}^{(r)}(v)} \frac{1}{\sqrt{\tilde{d}_u \tilde{d}_v}} \mathbf{W}_H^{(r)} \mathbf{h}^{(l-1)}_u \right)
\end{equation}

\textbf{2. Heterophilic Channel ($\mathbf{h}^{X}_v$):} Isolates anomalous structural spikes (e.g., massive outbound followings to unrelated nodes) using permutation-invariant max-pooling over the \textit{unrefined} graph $A$:
\begin{equation}
    \mathbf{h}^{X}_v = \max_{u \in \mathcal{N}^{(r)}(v)} \left( \text{MLP}_X (\mathbf{h}^{(l-1)}_u) \right)
\end{equation}

\textbf{3. Self-Identity Channel ($\mathbf{h}^{S}_v$):} Provides a residual pathway to prevent feature wash-out:
\begin{equation}
    \mathbf{h}^{S}_v = \mathbf{W}_S \mathbf{h}^{(l-1)}_v
\end{equation}

These three orthogonal representations are fused using a node-specific attention mechanism:
\begin{equation}
    \alpha_k = \frac{\exp(\mathbf{q}^T (\mathbf{W}_{att} \mathbf{h}^k_v))}{\sum_{c \in \{H, X, S\}} \exp(\mathbf{q}^T (\mathbf{W}_{att} \mathbf{h}^c_v))}
\end{equation}
\begin{equation}
    \mathbf{h}^{(l)}_v = \sum_{k \in \{H, X, S\}} \alpha_k \mathbf{h}^k_v
\end{equation}

\subsection{Contrastive Prototype Disentanglement (CPD)}
Even after careful aggregation, the learned latent representation $\mathbf{h}_v$ may still entangle behavioral bot traits with benign social positioning artifacts. To resolve this, CPD projects $\mathbf{h}_v$ into two distinct sub-spaces: a behavioral signature code $\mathbf{z}^b_v$ and a social positioning code $\mathbf{z}^s_v$.

To enforce disentanglement, we minimize the mutual information (MI) between the two codes using an upper-bound orthogonal penalty:
\begin{equation}
    \mathcal{L}_{MI} = \frac{1}{N} \sum_{v \in \mathcal{V}} \left\| (\mathbf{z}^b_v)^T \mathbf{z}^s_v \right\|_F^2
\end{equation}

Concurrently, we align the behavioral code $\mathbf{z}^b_v$ with global class prototypes $\mathbf{P}_y \in \mathbb{R}^{d'}$ (where $y \in \{0,1\}$) using a Supervised InfoNCE loss:
\begin{equation}
    \mathcal{L}_{NCE} = - \frac{1}{N} \sum_{v \in \mathcal{V}} \log \frac{\exp( (\mathbf{z}^b_v)^T \mathbf{P}_{y_v} / \tau_{nce} )}{\sum_{k \in \{0,1\}} \exp( (\mathbf{z}^b_v)^T \mathbf{P}_k / \tau_{nce} )}
\end{equation}
This forces the model to strictly classify nodes based on their pure behavioral signature, stripped of adversarial topological noise.

\subsection{Hierarchical Forensic Explanation (HFE)}
Finally, to satisfy the critical requirement of platform transparency, the HFE module generates multi-level forensic evidence justifying the classification. 

At the \textbf{Instance Level}, we learn a soft edge mask $M \in [0,1]^{|\mathcal{E}|}$ that maximizes the mutual information between the original prediction and the prediction made using only the masked subgraph $G_S$:
\begin{equation}
    \min_{M} - \sum_{c=1}^C P(Y=c|\mathcal{G}, X) \log P(Y=c|\mathcal{G} \odot M, X) + \lambda \|M\|_1
\end{equation}

At the \textbf{Macro Level}, we apply the Louvain community detection algorithm to the bot-induced subgraph defined by the top-$K$ edges in $M$, successfully surfacing dense, coordinated botnet rings and calculating diagnostic heuristic motifs (e.g., star-formation density).
